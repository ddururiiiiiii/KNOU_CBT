<h1>방통대 CBT</h1>
 <ul>
     <li>나는 방통대 재학중이다. 기말시험 준비를 위해 기출문제(PDF)를 다운 받아 
       여러 번 풀어보다가 '최강 자격증 기출문제 전자문제집 CBT' (https://www.comcbt.com/) 와 같은 웹 애플리케이션 있으면 좋겠다는 생각을 했다.
       그래서 시작한 프로젝트!
     </li>
 </ul>

<h2>관심사 및 목표</h2>
